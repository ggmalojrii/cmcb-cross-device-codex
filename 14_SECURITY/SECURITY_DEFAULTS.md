# Security Defaults

- SSH keys only.
- Use Tailscale/private networking.
- Close public SSH after Tailscale works.
- Do not expose test servers publicly by default.
- Use Git branches before Codex edits.
- Sync logs/artifacts, not secrets.
- API keys only in environment variables.
- No secrets in CMCB packets/logs.
- Downloads quarantine before execution.
- Admin installs require user approval and logs.
