CMCB Housekeeping Console

A queue-based file management surface for safe add/edit/delete work. It is approval-gated, root-limited, and designed to keep the public site from becoming a raw shell.

What It Does

  • Stages file requests for the repo and handoff surface.
  • Lets ChatGPT build request packets instead of guessing edits.
  • Requires approval before destructive changes are applied.

Allowed Roots

  • repo_clone
  • shared/CMCB-Shared/handoffs/chatgpt-online

Not Allowed

  • Secrets, private keys, cloud tokens, or .env files.
  • Arbitrary shell commands.
  • Anything outside the two safe roots above.

Queue Flow

  • Build a request packet here.
  • Save it into the pending queue.
  • Move approved packets to the approved queue.
  • Let the housekeeping agent process approved work.

How To Use It

  1. Choose the root and operation.
  2. Fill in the path and content if needed.
  3. Keep Dry run only checked until you want a real change.
  4. Use Copy JSON or Download JSON to stage the request packet.
  5. Move approved requests into the approved queue for the housekeeping agent.
