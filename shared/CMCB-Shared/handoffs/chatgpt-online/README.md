# CMCB ChatGPT Online Handoff

This folder is the only intended public surface for temporary ChatGPT/browser review.

It is sanitized:

- No credentials
- No private keys
- No cloud tokens
- No `.env` files
- No filesystem browsing outside this folder

Use `index.html` for human review and `status.json` for machine-readable status.

For laptop setup, download `laptop_onboarding.ps1` and run it in PowerShell.
